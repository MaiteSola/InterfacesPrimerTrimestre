Ten cuidado con la ruta
sass -w .\scss\:.\css\
Para copiar todas las carpetas


Variables con $
!global para sobreponer
!default para poner por defecto

En este formato un ejemplo:
$micolor: #00c4c7; 
$micolor: #c70000 !default; 

h2 {
   
   $micolor: #c70000 !global;    
   color: $micolor; 
}

h1 {    
   color: $micolor; 
}

Documentación de SASS: 
●Números (ejemplo: 1.2, 13, 10px) 
●Strings, Cadenas de texto con o sin comillas simples o dobles (ejemplo "foo", bar', baz) 
●Valores lógicos o booleanos (ejemplo true, false) ●Valores nulos (ejemplo null) 
●Listas de valores, separados por espacios en blanco o comas (ejemplo 1.5em 1em 0 2em, Helvetica, Arial, sans-serif) 
●Pares formados por una clave y un valor separados por: (ejemplo (key1:value1, key2: value2))

// el > sirve para meternos dentro de esa sección por ejemplo.
article {

   section>h3{
      line-height: 1em;
   }
}

//El + es el hermano de sección.
// <section></section>
// <p></p> Ese p es hermano. Si estuviera dentro sería hijo.
section+p{

   color:azure;

}

 todos los hermanos de la sección sería así: section~p{}
section~p{

   color:azure;

}

   padding: ceil(1.2em);

   border-bottom: (random(10)+px) dotted #c80060;
